Codificación HTML (conjuntos de caracteres)

Para mostrar una página HTML correctamente, un navegador web debe saber qué conjunto de caracteres utilizar.

El atributo charset HTML

El conjunto de caracteres se especifica en la <meta> etiqueta:

<meta charset="UTF-8">

La especificación HTML5 alienta a los desarrolladores web a utilizar el conjunto de caracteres UTF-8.

¡UTF-8 cubre casi todos los caracteres y símbolos del mundo!

El conjunto de caracteres ASCII

ASCII fue el primer estándar de codificación de caracteres para la web. Definía 128 caracteres diferentes que podían utilizarse en Internet:

  • Letras inglesas (AZ)
  • Números (0-9)
  • Caracteres especiales como ! $ + - ( ) @ < >.

El conjunto de caracteres ANSI

ANSI (Windows-1252) fue el conjunto de caracteres original de Windows:

  • Idéntico a ASCII para los primeros 127 caracteres
  • Caracteres especiales del 128 al 159
  • Idéntico a UTF-8 de 160 a 255
<meta charset="Windows-1252">

El conjunto de caracteres ISO-8859-1

ISO-8859-1 era el conjunto de caracteres predeterminado para HTML 4. Este conjunto de caracteres admitía 256 códigos de caracteres diferentes. HTML 4 también admitía UTF-8.

  • Idéntico a ASCII para los primeros 127 caracteres
  • No utiliza los caracteres del 128 al 159
  • Idéntico a ANSI y UTF-8 de 160 a 255
<meta http-equiv="Content-Type" content="text/html;charset=ISO-8859-1">

Ejemplo de HTML 5

<meta charset="ISO-8859-1">

El conjunto de caracteres UTF-8

Es idéntico a ASCII para los valores de 0 a 127. No utiliza los caracteres del 128 al 159. Es idéntico a ANSI y 8859-1 de 160 a 255. Continúa desde el valor 256 hasta 10 000 caracteres.

<meta charset="UTF-8">

Codificación de URL

Las URL solo se pueden enviar a través de Internet utilizando el conjunto de caracteres ASCII. Si una URL contiene caracteres que no pertenecen al conjunto ASCII, es necesario convertirla.

La codificación de URL convierte caracteres que no son ASCII en un formato que puede transmitirse a través de Internet. Reemplaza los caracteres no ASCII con un "%" seguido de dígitos hexadecimales. Los espacios se reemplazan con un signo más (+) o %20.
