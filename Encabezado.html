HTML - El elemento Head

El elemento HTML <head> es un contenedor para los metadatos y otros elementos importantes como <title>, <style>, <meta>, <link>, <script> y <base>.

El elemento HTML <head>

El <head> elemento es un contenedor de metadatos (datos sobre datos) y se coloca entre las etiquetas <html> y <body>.

El elemento HTML <style>

El <style> elemento se utiliza para definir información de estilo para una página HTML.

El elemento HTML <link>

El <link> elemento define la relación entre el documento actual y un recurso externo, como una hoja de estilo externa.

El elemento HTML <meta>

El <meta> elemento se utiliza para especificar metadatos como el conjunto de caracteres, descripción de la página, palabras clave, autor, etc.

El elemento HTML <script>

El <script> elemento se utiliza para definir JavaScript del lado del cliente en una página web.

El elemento HTML <base>

El <base> elemento especifica la URL base y/o el destino de todas las URL relativas en una página.
