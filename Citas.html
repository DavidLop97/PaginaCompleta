Elementos de cita y cita HTML

En este capítulo repasaremos los elementos HTML <blockquote>, <q>, <abbr>, <address>, <cite> y <bdo>.

HTML <blockquote> para citas

El elemento HTML <blockquote> define una sección que se cita de otra fuente. Los navegadores generalmente sangran los elementos <blockquote>.

HTML <q> para citas breves

La etiqueta HTML <q> define una cita corta. Los navegadores normalmente insertan comillas alrededor de las citas.

HTML <abbr> para abreviaturas

La etiqueta HTML <abbr> define una abreviatura o un acrónimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Marcar abreviaturas puede proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda.

Consejo: utilice el atributo de título global para mostrar la descripción de la abreviatura/acrónimo cuando pase el mouse sobre el elemento.

Ejemplo:


HTML <address> para información de contacto

La etiqueta HTML <address> define la información de contacto del autor/propietario de un documento o artículo.

La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, cuenta de redes sociales, etc.

El texto del <address> elemento normalmente se muestra en cursiva y los navegadores siempre agregarán un salto de línea antes y después del <address> elemento.

Ejemplo:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

HTML <cite> para el título del trabajo

La etiqueta HTML <cite> define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).

El texto del <cite> elemento normalmente se muestra en cursiva.
