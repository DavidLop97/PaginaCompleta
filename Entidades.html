Guía de estilo HTML

Entidades HTML

Los caracteres reservados en HTML deben reemplazarse por entidades:

  • < (menor que) = <
  • > (mayor que) = >

Entidades de caracteres HTML

Algunos caracteres están reservados en HTML. Para mostrarlos correctamente, se pueden usar nombres de entidad o números de entidad:

  • Los nombres de entidades se ven así: &entity_name;
  • Los números de entidad se ven así: &#entity_number;

Espacio irrompible

Un espacio indivisible es un espacio que no se interrumpe y no forma una nueva línea. Se utiliza para mantener juntas palabras que de otro modo podrían separarse:

  • Ejemplos:
  • 10 km/h
  • 22:00 hrs

Otro uso común es evitar que los navegadores trunquen los espacios en páginas HTML.
