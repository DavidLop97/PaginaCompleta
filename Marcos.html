Marcos HTML

Un iframe HTML se utiliza para mostrar una página web dentro de otra página web.

Sintaxis de iframe en HTML

La etiqueta HTML <iframe> especifica un marco en línea.

Un marco en línea se utiliza para incrustar otro documento dentro del documento HTML actual.

Ejemplo básico:

<iframe src="https://www.example.com" title="Ejemplo" height="200" width="300"></iframe>

Iframe - Establecer altura y ancho

Utiliza los atributos height y width para especificar el tamaño del iframe.

La altura y el ancho se pueden especificar en píxeles o utilizando el estilo CSS:

<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Ejemplo de Iframe"></iframe>

Iframe - Eliminar el borde

Para eliminar el borde predeterminado del iframe, usa el atributo style con border:none;:

<iframe src="demo_iframe.htm" style="border:none;" title="Ejemplo de Iframe sin borde"></iframe>

Iframe - Objetivo de un enlace

Un iframe puede utilizarse como marco de destino para un enlace utilizando el atributo target del enlace:

<iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe>

<a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a>
