Imágenes HTML

Las imágenes pueden mejorar el diseño y la apariencia de una página web.

Sintaxis de imágenes HTML

La <img> etiqueta HTML se utiliza para incrustar una imagen en una página web.

Técnicamente, las imágenes no se insertan en una página web, sino que se vinculan a páginas web. La <img> etiqueta crea un espacio de almacenamiento para la imagen a la que se hace referencia.

La <img> etiqueta está vacía, contiene solo atributos y no tiene una etiqueta de cierre.

Sintaxis

<img src="url" alt="alternatetext">

El atributo src

El atributo requerido src especifica la ruta (URL) a la imagen.

Nota: Asegúrate de que la imagen permanezca en el mismo lugar en relación con la página web para evitar que los visitantes vean un icono de enlace roto.

El atributo alt

El atributo requerido alt proporciona un texto alternativo para una imagen, útil si el usuario no puede verla por alguna razón.

El valor del atributo alt debe describir la imagen.

Tamaño de la imagen: ancho y alto

Puedes usar el atributo style para especificar el ancho y la altura de una imagen:

Ejemplo

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

Alternativamente, puedes utilizar los atributos width y height:

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Siempre especifica el ancho y la altura de una imagen para evitar parpadeos mientras se carga la página.

Imágenes en otra carpeta

Si tienes imágenes en una subcarpeta, incluye el nombre de la carpeta en el atributo src:

Ejemplo

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
