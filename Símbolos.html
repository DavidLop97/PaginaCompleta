Entidades de símbolos HTML

Se pueden agregar símbolos o letras que no están presentes en el teclado al HTML mediante entidades.

Entidades de símbolos HTML

Muchos símbolos matemáticos, técnicos y monetarios no están presentes en un teclado normal. Para agregar estos símbolos a una página HTML, se puede utilizar el nombre de la entidad o el número de entidad (referencia decimal o hexadecimal) para el símbolo:

Ejemplo:

            Mostrar el símbolo del euro:
            I will display €
            I will display €
            I will display €

Se mostrará como:

Mostraré €

Mostraré €

Mostraré €

Algunos símbolos matemáticos admitidos por HTML

Char Number Entity Description
∀ ∀ &forall; For all
∂ ∂ &part; Partial differential
∃ ∃ &exist; There exists
∅ ∅ &empty; Empty sets
∇ ∇ &nabla; Nabla
∈ ∈ &isin; Element of
∉ ∉ &notin; Not an element of
∋ ∋ &ni; Contains as member
∏ ∏ &prod; N-ary product
∑ ∑ &sum; N-ary summation
