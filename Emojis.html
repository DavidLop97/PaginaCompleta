¿Qué son los emojis?

Los emojis parecen imágenes o íconos, pero son letras (caracteres) del conjunto de caracteres UTF-8 (Unicode).

El atributo charset HTML

Para mostrar una página HTML correctamente, un navegador web debe conocer el conjunto de caracteres utilizado en la página. Esto se especifica en la <meta> etiqueta:

<meta charset="UTF-8">

Caracteres UTF-8

Muchos caracteres UTF-8 no se pueden escribir en un teclado, pero siempre se pueden mostrar mediante números (llamados números de entidad).

Personajes Emoji

Los emojis también son caracteres del alfabeto UTF-8:

  • 😄 es 128516
  • 😍 es 128525
  • 💗 es 128151
