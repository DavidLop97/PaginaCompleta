La etiqueta HTML <script>

La etiqueta HTML <script> se utiliza para definir un script del lado del cliente (JavaScript).

El <script> elemento contiene declaraciones de script o apunta a un archivo de script externo a través del atributo src.

Un uso común de JavaScript es la manipulación de elementos HTML utilizando document.getElementById() para seleccionar elementos por su ID, como se muestra en el siguiente ejemplo:

<script>
            document.getElementById("demo").innerHTML = "Hello JavaScript!";
            </script>

Ejemplos de JavaScript

A continuación se muestran algunos ejemplos adicionales de lo que JavaScript puede hacer:

Cambiar el contenido de un elemento HTML

<script>
            document.getElementById("demo").innerHTML = "Hello JavaScript!";
            </script>

Cambiar estilos de un elemento HTML

<script>
            document.getElementById("demo").style.fontSize = "25px";
            document.getElementById("demo").style.color = "red";
            document.getElementById("demo").style.backgroundColor = "yellow";
            </script>

Cambiar atributos de un elemento HTML

<script>
            document.getElementById("image").src = "picture.gif";
            </script>

La etiqueta HTML <noscript>

La etiqueta <noscript> define un contenido alternativo que se mostrará a los usuarios que tienen los scripts deshabilitados en su navegador o usan un navegador que no admite scripts:

<script>
            document.getElementById("demo").innerHTML = "Hello JavaScript!";
            </script>
            <noscript>Sorry, your browser does not support JavaScript!</noscript>
