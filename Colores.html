Colores HTML

Los colores HTML se especifican con nombres de color predefinidos o con valores RGB, HEX, HSL, RGBA o HSLA.

Nombres de colores

En HTML, un color se puede especificar utilizando un nombre de color:


HTML admite 140 nombres de colores estándar.

Color de fondo

Puedes establecer el color de fondo para los elementos HTML:
