Comparación entre HTML y XHTML

XHTML es una versión de HTML más estricta y basada en XML.

¿Qué es XHTML?

  • XHTML significa Lenguaje de Marcado de Hipertexto Extensible.
  • XHTML es una versión de HTML más estricta y basada en XML.
  • XHTML es HTML definido como una aplicación XML.
  • XHTML es compatible con todos los navegadores principales.

¿Por qué XHTML?

XML es un lenguaje de marcado donde todos los documentos deben estar bien formados. XHTML se desarrolló para hacer que HTML fuera más extensible y flexible para trabajar con otros formatos de datos como XML. Además, XHTML tiene un manejo de errores mucho más estricto en comparación con HTML.

Diferencias más importantes con respecto a HTML

  • <!DOCTYPE> es obligatorio.
  • El atributo xmlns en <html> es obligatorio.
  • Los elementos <html>, <head>, <title> y <body> son obligatorios.
  • Los elementos siempre deben estar correctamente anidados.
  • Los elementos siempre deben estar cerrados.
  • Los elementos vacíos también deben estar cerrados.
  • Los elementos y los nombres de atributos deben estar en minúsculas.
  • Los valores de los atributos deben estar entre comillas.
  • Está prohibida la minimización de atributos.
